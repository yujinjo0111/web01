web이란?

월드 와이드 웹 ( WWW 일반적으로 알려진), 웹 ,입니다 정보 시스템 문서 및 기타 웹 자원 에 의해 식별되는 범용 리소스 로케이터 등 (URL을,https://example.com/) 하이퍼링크 로 연결될 수 있으며 인터넷을 통해 액세스할 수 있습니다 . [1] [2] 웹의 리소스는 HTTP( Hypertext Transfer Protocol) 를 통해 전송 되고 사용자 는 웹 브라우저 라는 소프트웨어 응용 프로그램에서 액세스할 수 있으며 웹 서버 라는 소프트웨어 응용 프로그램에 의해 게시됩니다 . World Wide Web은 어떤 형태로든 웹보다 20년 이상 앞서고 웹이 구축되는 기술을 기반으로 하는 인터넷과 동의어가 아닙니다. 영국 과학자 Tim Berners-Lee 는 1989년에 월드 와이드 웹을 발명했습니다. 그는 스위스 제네바 근처 CERN 에서 근무하는 동안 1990년에 최초의 웹 브라우저를 작성했습니다 . [3] [4] 브라우저는 1991년 1월부터 CERN 외부의 다른 연구 기관에 출시되었으며, 그 다음 1991년 8월에 일반 대중에게 공개되었습니다. 웹은 일반 사용 웹 사이트 가 등장하기 시작한 1993~1994년에 일상적 으로 사용 되기 시작했습니다. 사용 가능하게 됩니다. [5] 월드 와이드 웹은 정보화 시대 의 중심이었으며 수십억 명의 사람들이 인터넷에서 상호 작용하기 위해 사용하는 주요 도구입니다. [6] [7] [8] [9] [10] 웹 리소스는 모든 유형의 다운로드된 미디어일 수 있지만 웹 페이지 는 HTML( Hypertext Markup Language) 형식 의 하이퍼 텍스트 문서 입니다. [11] 특수 HTML 구문은 사용자 가 다른 웹 리소스 로 이동할 수 있도록 하는 URL이 포함된 하이퍼링크 를 표시 합니다. 텍스트 외에도 웹 페이지에는 멀티미디어 콘텐츠 의 페이지 또는 스트림을 렌더링하기 위해 사용자의 웹 브라우저 에 표시되거나 내부적으로 실행되는 이미지 , 비디오 , 오디오 및 소프트웨어 구성 요소에 대한 참조가 포함될 수 있습니다 . 공통 테마와 일반적으로 공통 도메인 이름 을 가진 여러 웹 리소스가 웹사이트를 구성 합니다. 웹사이트는 사용자의 컴퓨터에서 실행되는 웹 브라우저에서 인터넷을 통해 만들어진 요청에 응답하는 프로그램인 웹 서버를 실행하는 컴퓨터에 저장 됩니다. 웹사이트 콘텐츠는 게시자가 제공하거나 사용자 생성 콘텐츠 에서 대화식으로 제공할 수 있습니다 . 웹사이트는 정보, 오락, 상업 및 정부 목적으로 제공됩니다. .
